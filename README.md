# Carla.agenciayaar – Tutorial

Site de ajuda oficial para criadoras da **Carla.agenciayaar**.

## Objetivo

Guia passo a passo para ensinar as criadoras a instalar o app, cadastrar e ativar a conta corretamente, seguindo o tutorial oficial.

## Descrição

Site estático com accordion interativo, visual feminino e acolhedor, 100% responsivo para celular. Cada etapa do tutorial está organizada em blocos expansíveis para facilitar a leitura.

## Como abrir localmente

1. Faça o download ou clone este repositório
2. Abra o arquivo `index.html` diretamente no navegador

## Como publicar na Vercel

1. Acesse [vercel.com](https://vercel.com)
2. Clique em **Add New Project**
3. Importe este repositório do GitHub
4. Clique em **Deploy** — pronto!
5. Para usar domínio próprio, vá em **Settings → Domains** e adicione seu domínio

## Como editar os tópicos

Abra o arquivo `index.html` e localize os blocos `<article class="step-card">`. Cada bloco é um passo do tutorial. Edite o texto dentro de `<div class="step-body">`.

## Estrutura de arquivos

```
/index.html      → Página principal com todo o conteúdo
/style.css       → Estilos visuais
/script.js       → Accordion e interações
/favicon.svg     → Ícone do site
/vercel.json     → Configuração de deploy
/assets/img/     → Imagens do tutorial
```

## Observação

O conteúdo segue o PDF oficial do tutorial da agência. Não altere a ordem dos passos sem consultar o material oficial.
